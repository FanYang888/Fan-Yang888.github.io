常见状态码

2XX一般表示请求成功

状态码 2xx 说明
200 请求被正确处理
204 No content 请求成功，但响应报文不含实体的主体部分
205 Reset Content 请求成功，但是与 204 响应不同在于要求请求方重置内容
206 Partial Content 进行范围请求

3XX一般表示重定向、304除外

状态码3xx 说明
301 永久重定向
302（http1.0） 临时重定向
303（http1.1） 明确表示客户端应当采⽤get⽅法获取资源，把POST请求变为GET请求进⾏重定向
307 （http1.1） 遵照浏览器标准，不会从POST变为GET
304 协商缓存

4XX一般表示客户端错误

状态码4xx 说明
400 bad request 请求报文存在语法错误
401 unauthorized 未认证、未授权
403 forbidden 被禁止访问
404 not found 服务器未找到资源

5XX一般表示服务端错误

状态码5xx 说明
500 internal sever error 服务器执行请求时发生错误
501 Not Implemented 服务器不支持当前请求的某个功能
502 Bad Gateway 网关或代理服务器接收到的请求无效，通常需要找网络管理员解决
503 Service Unavailable 服务器超负载或停机
504 Gateway Timeout(http1.1) 服务器请求超时
