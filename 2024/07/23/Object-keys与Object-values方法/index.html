Object.keys与Object.values方法及注意点

1.使用方法：
Object.keys(obj) 获取obj中的键

Object.values(obj) 获取obj中的值

2.注意点：

  • Object.keys(obj)返回的结果，如果键是数字，则顺序是按键数字从小到大的顺序，但会将数字转成字符串，如果是字符或‘ ’，按原顺序返回。
  • Object.keys(obj)返回的结果没有重复的。
  • Object.keys(obj)返回的结果,如果键是小数，则按照1中的字符顺序处理。
  • Object.values(obj)返回的结果顺序与Object.keys(obj)返回的键的顺序对应。
  • Object.values(obj)返回的结果，若存在相同的键，则该键的结果为后者。
